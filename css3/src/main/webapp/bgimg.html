background 배경

  1. background_color: color; 배경색상
  2. background_image: url("이미지 경로"); 배경 이미지

배경이미지

  • background-repeat : 배경이미지 반복;
    • repeat 기본값, 무한반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복 안함
  • background-size : 배경이미지 크기;
    • cover 이미지 하나로 영역 전체를 다 덮음. (이미지가 잘려나갈 수 있음. )
    • contain 이미지 하나를 온전하게 영역에 최대한 엎음. (이미지가 잘려나가지 않으나 빈공간나 반복이 있을 수 있음. )
    • Xn Yn 수평 수직 순서로 임의의 값 지정.
  • background-position : 배경이미지 위치;
    • left/center/right top/center/bottom 수평 수직 순서로 방향값을 조합
    • Xn Yn 수평 수직 순서로 임의의 값 지정
  • background-attachment : 배경이미지 고정여부;
    • scroll 자신의 스크롤바에 고정, 브라우저 스크롤바에는 스크롤
    • fixed 자신과 브라우저 스크롤바에 모두 고정.
    • local 자신과 브라우저 스크롤바에 모두 스크롤.
