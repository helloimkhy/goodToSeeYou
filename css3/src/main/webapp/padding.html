padding 요소의 내부 여백

테두리(border)를 기준으로 안쪽으로 사방에 적용 됨.

1. padding의 방향별 지정 속성

  • padding-top: n; 위에 n만큼 적용
  • padding-right: n; 우측에 n만큼 적용
  • padding-bottom: n; 아래에 n만큼 적용
  • padding-left: n; 좌측에 n만큼 적용

2. padding의 값 갯수별 지정법

  • padding : n; 사방에 같은 값을 동시 적용
  • padding : n n; 상하 좌우 순서로 적용
  • padding : n n n; 상 좌우 하 순서로 적용
  • padding : n n n n; 상 우 하 좌 순서로 적용
