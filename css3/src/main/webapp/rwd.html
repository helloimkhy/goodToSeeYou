rwd(responsive web design)

mobile first

viewport

뷰포트란 웹 페이지에서 사용자가 볼 수 있는 영역 <head> <meta name="viewport" content="width=device-width, initial-scale=1.0"> </head>

반응형 작업 Media Query의 문법

@media 조건1 and (조건2) { 선택자 {속성:속성값;} }

조건1의 장치 종류(Media Types)

  • all - 모든 장치(상관없음)
  • print - 프린터
  • screen - 화면으로 나타나는 컴퓨터(스마트폰,테블릿 포함)
  • speech - 음성낭독기

조건2의 종류

  • width and height of the viewport - 화면의 너비나 높이
  • width and height of the device - 장치의 너비나 높이
  • orientation (landscape or portrait) - 화면 방향(가로/세로)
  • resolution - 해상도
