border-radius 모서리 라운딩

  • border-radius:n; 양사방n으로 동일
  • border-radius:n n; 좌측상단과 우측하단이 첫번째, 우측상단과 좌측하단이 두번쨰로 서로 동일
  • border-radius:n n n n;
