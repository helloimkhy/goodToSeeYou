margin 요소의 외부 여백

테두리(border)를 기준으로 바깥으로 사방에 적용 됨.

1. margin의 방향별 지정 속성

  • margin-top: n; 위에 n만큼 적용
  • margin-right: n; 우측에 n만큼 적용
  • margin-bottom: n; 아래에 n만큼 적용
  • margin-left: n; 좌측에 n만큼 적용

2. margin의 값 갯수별 지정법

  • margin : n; 사방에 같은 값을 동시 적용
  • margin : n n; 상하 좌우 순서로 적용
    항상 정 중앙에 위치 하는 요소
    margin:n auto; (margin-left: auto; margin-right: auto;)
  • margin : n n n; 상 좌우 하 순서로 적용
  • margin : n n n n; 상 우 하 좌 순서로 적용

3. inline(a)의 margin적용

상하 margin 적용 불가, 인접요소와 좌우 margin은 이중으로 적용. (크기 적용 불가)
